Installation Complete

Knit is ready to use. Right-click any file or folder in Finder to access these items:

  • Knit Compress (ZIP) — produces a standard ZIP archive
  • Knit Compress (.knit) — produces Knit's high-speed .knit archive
  • Knit Extract — extracts ZIP or .knit archives

Look for these under the Quick Actions submenu in the right-click menu (they also appear in the Finder preview pane on the right when a file is selected).

If the items don't appear, open System Settings → Keyboard → Keyboard Shortcuts → Services → Files and Folders, and make sure the Knit entries are enabled — macOS groups both Services and Quick Actions under that pane.
For command-line usage, run knit --help.
